# Tabo

## Environment

### Operating System
macOS High Sierra (Version 10.13.3)

### Environment Variables
None

## Dependencies
- [node](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)
- [web-ext](https://www.npmjs.com/package/web-ext)

## Building

```
$ npm install
$ npm run build
```

## Running

```
$ cd dist
$ web-ext run --no-reload
```
