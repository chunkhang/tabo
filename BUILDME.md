# Tabo

## Environment

### Operating System
macOS High Sierra (Version 10.13.3)

### Environment Variables
None

## Dependencies
- node
- npm

## Building

```
$ npm install
$ npm run build
```

## Running

```
$ cd dist
$ web-ext run --no-reload
```
